# 文件上传模块详细设计

## 1. 模块概述
本模块提供通用文件上传能力，当前用于上传到 OSS 并返回访问地址。

## 2. 功能详细描述
1. 接收 Multipart 文件。
2. 调用 OSS 工具上传。
3. 返回上传后的 URL。

## 3. 接口设计
- `POST /upload`
- 参数：`file`（multipart）
- 返回：文件 URL

## 4. 数据结构设计
- Controller：`UploadController`
- 工具类：`AliOSSUtils`
- 返回结构：统一 `Result`

## 5. 业务流程设计
1. 前端上传文件。
2. 后端记录文件名日志。
3. 调用 OSS SDK 上传。
4. 返回 URL 给前端。

## 6. 异常处理设计
- 上传异常抛出 IOException。
- 建议增加统一异常拦截，将 IO 异常转标准错误码。

## 7. 安全设计
1. 建议增加文件类型白名单。
2. 建议限制文件大小与上传频率。
3. 建议启用恶意文件检测与签名 URL。
